# Hydrated Farmland

Hytale plugin that hydrates farmland close to water, just like Minecraft.

Uses a random tick based system with configurable tick speed.

Distance from water required is also configurable.

## Command Usage

```
/hydratedfarmland (alias: hf)
    config (alias: cfg)
        ticks \[--set <integer>]
        distance \[--set <integer>]
```
